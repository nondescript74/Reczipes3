# Cooking Mode - Quick Reference Card

## 🔥 What is Cooking Mode?

A hands-free cooking interface that displays up to two recipes side-by-side, perfect for following complex recipes or timing multiple dishes.

---

## 📱 Layout by Device

| Device | Layout | Navigation |
|--------|--------|------------|
| **iPad/Mac** | Side-by-side with vertical divider | Both recipes visible at once |
| **iPhone** | Swipeable pages | Swipe left/right, dots show current page |

---

## ⚡ Key Features

### 🔄 Dual Recipe View
- View up to 2 recipes simultaneously
- Empty slots show "+" button to select recipes
- Each slot is independent
- Perfect for main dish + side dish

### 👁️ Keep Awake
- **Icon**: Eye (on) / Eye with slash (off)
- **Location**: Top-right corner
- **Function**: Prevents screen from sleeping
- **Tip**: Enable before cooking, disable when done

### 🎯 Recipe Controls (Floating Buttons)
| Button | Icon | Action |
|--------|------|--------|
| **Swap** | ↻ | Choose different recipe |
| **Clear** | ✕ | Remove recipe from slot |

### 💾 Auto-Save
- Selected recipes saved automatically
- Keep Awake preference remembered
- Current page saved (iPhone)
- Session persists across app launches

---

## 🚀 Getting Started

1. **Open Cooking Mode** from main tab bar
2. **Tap "+"** on empty slot
3. **Select recipe** from your collection
4. **Enable Keep Awake** (eye icon)
5. **Start cooking!**

---

## 💡 Pro Tips

### Choosing Recipes
- Tap "+" on any empty slot
- Tap ↻ (swap) to change existing recipe
- Search by name in recipe picker
- Select and sheet auto-dismisses

### Managing Slots
- Use 1 recipe for simple cooking
- Use 2 recipes for complex meals
- Clear slots you don't need
- Mix and match as needed

### Device-Specific
- **iPad**: Resize screen to see more of each recipe
- **iPhone**: Swipe quickly between recipes
- **Both**: Pinch to zoom recipe text
- **All**: Rotate device for best view

### Screen Management
- Enable Keep Awake before starting
- Disable when taking breaks to save battery
- Screen stays on even during calls (iOS feature)
- Automatically disabled when exiting mode

## 🔍 Common Tasks

```
Empty Slot → Tap "+" → Select Recipe → Done
```

### Add Second Recipe
```
Other Empty Slot → Tap "+" → Select Recipe → Done
```

### Change Recipe
```
Tap ↻ on Recipe → Select Different Recipe → Done
```

### Remove Recipe
```
Tap ✕ on Recipe → Slot Becomes Empty
```

```
Swipe Left/Right → Check Dots → View Recipe
```

## ⚠️ Troubleshooting

### Recipe Picker Keeps Appearing
- **Fixed**: Latest version auto-dismisses
- **If issue persists**: Force quit and reopen app

### Screen Still Sleeps
- **Check**: Keep Awake is ON (eye icon)
- **Verify**: Icon shows open eye, not slashed
- **Setting**: Not affected by system settings
- **Note**: Calls will still lock screen (iOS)

### Recipes Don't Save
- **Normal**: Auto-saves on changes
- **Wait**: Give it 2-3 seconds
- **Return**: Check if recipes restored
- **If not**: Check storage space

### Layout Looks Wrong
- **iPad**: Rotate device and back
- **iPhone**: Swipe to trigger refresh
- **Both**: Force quit app and reopen
- **Check**: iOS version is up-to-date

### Can't Find Recipe
- **Picker**: Use search at top
- **Check**: Recipe is actually saved
- **Verify**: Recipe has a title
- **Try**: Scroll down full list
