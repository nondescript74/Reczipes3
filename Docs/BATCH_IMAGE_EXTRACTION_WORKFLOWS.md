## Detailed Processing Flow

```
START
  в”Ӯ
  в”ңв”ҖвҶ’ [User selects N images]
  в”Ӯ
  в”ңв”ҖвҶ’ [User chooses crop option]
  в”Ӯ     вҖў ON: Will prompt for each image
  в”Ӯ     вҖў OFF: Process all as-is
  в”Ӯ
  в”ңв”ҖвҶ’ [Start batch extraction]
  в”Ӯ
  в”ңв”ҖвҶ’ Initialize:
  в”Ӯ     вҖў totalToExtract = N
  в”Ӯ     вҖў currentProgress = 0
  в”Ӯ     вҖў successCount = 0
  в”Ӯ     вҖў failureCount = 0
  в”Ӯ     вҖў remainingAssets = [all N assets]
  в”Ӯ
  в”ңв”ҖвҶ’ FOR EACH asset IN remainingAssets:
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ Check if stopped вҶ’ BREAK
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ WHILE paused:
  в”Ӯ     в”Ӯ     в””в”ҖвҶ’ Wait 0.1s
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ Load image from Photos
  в”Ӯ     в”Ӯ     в”ңв”Җ Success вҶ’ Continue
  в”Ӯ     в”Ӯ     в””в”Җ Failure вҶ’ Log error, increment failureCount, CONTINUE
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ IF crop enabled:
  в”Ӯ     в”Ӯ     в”Ӯ
  в”Ӯ     в”Ӯ     в”ңв”ҖвҶ’ Show image preview
  в”Ӯ     в”Ӯ     в”Ӯ
  в”Ӯ     в”Ӯ     в”ңв”ҖвҶ’ Ask: "Skip or Crop?"
  в”Ӯ     в”Ӯ     в”Ӯ     в”ңв”Җ Skip вҶ’ Use original
  в”Ӯ     в”Ӯ     в”Ӯ     в””в”Җ Crop вҶ’ Show crop UI
  в”Ӯ     в”Ӯ     в”Ӯ           в”ңв”Җ Cropped вҶ’ Use cropped
  в”Ӯ     в”Ӯ     в”Ӯ           в””в”Җ Cancelled вҶ’ Use original
  в”Ӯ     в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ Extract recipe via API
  в”Ӯ     в”Ӯ     в”ңв”Җ Success:
  в”Ӯ     в”Ӯ     в”Ӯ   в”ңв”ҖвҶ’ Save recipe to SwiftData
  в”Ӯ     в”Ӯ     в”Ӯ   в”ңв”ҖвҶ’ Save image to disk
  в”Ӯ     в”Ӯ     в”Ӯ   в”ңв”ҖвҶ’ Create image assignment
  в”Ӯ     в”Ӯ     в”Ӯ   в””в”ҖвҶ’ Increment successCount
  в”Ӯ     в”Ӯ     в”Ӯ
  в”Ӯ     в”Ӯ     в””в”Җ Failure:
  в”Ӯ     в”Ӯ         в”ңв”ҖвҶ’ Log error with details
  в”Ӯ     в”Ӯ         в””в”ҖвҶ’ Increment failureCount
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ Increment currentProgress
  в”Ӯ     в”Ӯ
  в”Ӯ     в”ңв”ҖвҶ’ Remove from remainingAssets
  в”Ӯ     в”Ӯ
  в”Ӯ     в””в”ҖвҶ’ IF currentProgress % 10 == 0:
  в”Ӯ           в””в”ҖвҶ’ Pause 0.5s (show progress)
  в”Ӯ
  в””в”ҖвҶ’ Show completion alert
        в””в”ҖвҶ’ END
```
