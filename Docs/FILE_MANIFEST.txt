# Claude Recipe Extractor - File Manifest

## Production Files (Add to your Xcode project)

### Core Implementation (Required)
1. ImagePreprocessor.swift (3.9 KB)
   - Image enhancement for better OCR
   - Grayscale, contrast, sharpening filters
   - Comparison view support

2. ClaudeAPIClient.swift (11 KB)
   - Claude API communication
   - Request/response handling
   - JSON parsing and conversion
   - Error handling

3. RecipeExtractorViewModel.swift (2.3 KB)
   - MVVM state management
   - Published properties for reactive UI
   - Extraction flow coordination

4. RecipeExtractorView.swift (11 KB)
   - Main UI for recipe extraction
   - Image picker integration
   - Preprocessing controls
   - Navigation and error display

5. RecipeDetailView.swift (12 KB)
   - Formatted recipe display
   - Multiple section support
   - Share functionality
   - Color-coded notes

6. ImagePicker.swift (1.5 KB)
   - Camera/photo library wrapper
   - UIKit bridge for SwiftUI

7. RecipeExtractorConfig.swift (6.4 KB)
   - Centralized configuration
   - API key management (Keychain, Environment)
   - Feature flags
   - Preprocessing settings

## Installation Order

1. Add core files (1-7) to Xcode project
2. Update Info.plist with camera/library permissions
3. Configure API key (see QUICK_START.md)
4. Optional: Add ExampleAppIntegration.swift for reference
5. Optional: Add RecipeExtractorTests.swift to test target

## Minimum Required Files

For basic functionality, you only need:
- ImagePreprocessor.swift
- ClaudeAPIClient.swift  
- RecipeExtractorViewModel.swift
- RecipeExtractorView.swift
- RecipeDetailView.swift
- ImagePicker.swift

You can skip RecipeExtractorConfig.swift if you directly pass API key 
and use default settings.

## File Dependencies

  └─ Used by: ClaudeAPIClient, ViewModel
